    • Deleting/ Restoring a User in Microsoft 365

    Prerequisites

    Logged into M365 admin


    • On the M365 admin dashboard, click the menu button
    • Click the Users dropdown
    • Click on Active Users


    • Locate the User you wish to delete
    • Click on their name and a window will popup


    • Click the Delete User button located at the top right


    • A deleted User's profile can be restored within 30 days after deletion
    • Other User's can access the deleted Users' Emails and OneDrive by checking the last two boxes


    • Click the Delete User button once completed


    • When the User is deleted, a message will popup
    • The User would also be deleted from the Active Users list


    • The deleted User can be found in the Deleted Users section


    • The User can be restored by clicking on the User's name
    • A popup will appear and the Restore User button would be at the bottom


    • Once restored, the User will appear on the Active Users list
    • However, they will not have a license
    • The User would have to be assigned a license manually by editing their profile
