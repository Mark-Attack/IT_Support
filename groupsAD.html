    • Creating a group in AD and Adding a User

    Prerequisites

    You're logged in to a domain-joined machine

    You have permissions to create groups

    Active Directory Users and Computers (ADUC) is installed


    Steps:

    • In windows searchbar type adm and click Windows Administrative Tools
    • Open Active Directory Users and Computers


    • navigate to the Organizational Unit (OU) where you want the group created
    • right click then click: new -> group


    • provide a name for the group
    • configure Group Scope and Group Type based on company's requirements
    • click OK to create the group


    • right click on newly created group and click properties
    • The group's properties window would appear
    • navigate to the members tab (notice no members are currently in this group)
    • click Add to add a User to this group


    • type the user's name into the search box then click Check Names
    • the user's name will appear with their email in brackets


    • click OK to add the User to the group
    • notice the User now appears in the group's members tab
