    Resolution

    Wi-Fi NOT working, Wired connection (Ethernet) working

    • It is an issue related to the specific device's Network Card
    • Ensure that the network adapter is enabled
    • Ensure that the user is trying to connect to the correct SSID
    • Use CMD command ipconfig /flushdns to clear the DNS cache on the PC
    • Use CMD commands ipconfig /release, then ipconfig /renew to clear the old IP address and request a new one from the DHCP server
    • Diconnect and reconnect to the Wi-fi
    • If a VPN is being used, then temporarily disconnect it and try connecting to the Wi-Fi (VPNs can block internet if misconfigured)
    • Restart the PC to refresh the network connection
    • Rare case: nework adapter is dead and needs replacing


    Both Wi-Fi and Wired connection (Ethernet) NOT working

    Find out the exact error message: Connected, no internet or Not connected

    • Connected, no internet means the network card and ethernet cable are both working and connecting to the network
    • Its the network that cannot reach the internet
    • Restart the moderm
    • Test other devices on the same network to see if the problem still exists
    • Contact ISP to determine if any outages occured

    • Not Connected means the device is not connected to any network
    • This is a local connection issue
    • For a wired connection, ensure that the ethernet cable is properly plugged in
    • Try testing with a different ethernet cable to rule out a faulty cable
    • Use CMD command ipconfig /flushdns to clear the DNS cache on the PC
    • Use CMD commands ipconfig /release, the ipconfig /renew to clear the old IP address and request a new one from the DHCP server
    • For a wireless connection, apply the network adapter steps mentioned above
