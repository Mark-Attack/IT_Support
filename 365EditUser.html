    • Editing a User's profile in Microsoft 365

    Prerequisites

    Logged into M365 admin


    • On the M365 admin dashboard, click the menu button
    • Click the Users dropdown
    • Click on Active Users


    • Locate the User's profile you wish to edit and click on their name


    • A window will popup with the selected User's profile
    • A User's information can be edited on the Account tab


    • A User's name and contact information can be editited on this page, along with many other things
    • A User's role can be changed via Manage Roles
    • A User can be added or removed from a group via Manage Groups
