    • Adding a New User to Microsoft 365

    Prerequisites

    Logged into M365 admin


    • On the M365 admin dashboard, click the menu button
    • Click the Users dropdown
    • Click on Active Users


    • Click the Add a user button


    • Add the User's basic information
    • Either check the box to auto create a password or create one manually for the User
    • Ensure that the box requiring the User to change the password at first sign is checked
    • click next once completed


    • Click the button to assign a product licenses to the User
    • Check the appropriate boxes
    • Then click the Apps dropdown


    • Check the Apps that you would like to assign to the User
    • Click next once completed


    • Optional Settings is where you give a User a specific role
    • Roles determine the level of privilige a User has
    • Select the appropriate role for that User
    • Click next once completed


    • Review information for any errors
    • Click Finish adding once completed


    • Once the User is added, you can close the window


    • The new User would now be shown in Active Users
