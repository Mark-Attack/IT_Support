    • Removing a User from a Group in AD

    Prerequisites

    You're logged in to a domain-joined machine

    Active Directory Users and Computers (ADUC) is installed


    Steps:

    • In windows searchbar type adm and click Windows Administrative Tools
    • Open Active Directory Users and Computers


    • navigate to the group in which the specific User is to be removed
    • right click on the group's name and click properties
    • navigate over to the members tab


    • click on the specific User to be removed and click remove
    • click apply, then OK
