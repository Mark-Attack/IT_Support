    • Resetting a User's password in Microsoft 365

    Prerequisites

    Logged into M365 admin


    • On the M365 admin dashboard, click the menu button
    • Click the Users dropdown
    • Click on Active Users


    • Locate the User whose password you with to reset
    • This can be done by clicking the Reset Password button above
    • Or by clicking the key next to a User's name


    • A window will popup with password settings
    • The User's new temp password can be auto generated by checking the first box
    • Checking the second box ensures that the User changes the temp password when they first login
    • Click Reset Password once completed
