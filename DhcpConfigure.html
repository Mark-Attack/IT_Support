    • How to Configure DHCP Server

    (Windows Server 2022)

    Prerequisites

    Windows Server is Installed

    Steps:

    • click the Windows button and click Server Manager


    • click add roles and features


    • For Installation Type, check Role-Based or feature based Installation


    • For Server Selection, select your server


    • For Server Roles, select DHCP Server
    • make sure the checkbox is ticked, then click Add Features


    • No Additional features needs to be installed so just click next for Features and DHCP Server
    • For Confirmation, click Install
    • Allow the installation to complete


    • When the installation is complete, click Complete DHCP Configuration


    • For Description, click next


    • Type the appropriate user credentials and click commit
