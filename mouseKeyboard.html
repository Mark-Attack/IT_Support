    • Mouse and/or Keyboard Not Working

    Potential Issues

    Bad USB Ports

    Dead Batteries (Wireless Devices)

    Outdated Drivers

    Dead Peripherals


    Resolution

    • Make sure the devices are plugged in correctly
    • For wireless devices, test or replace the batteries
    • Try plugging devices into a different port
    • Test with a known working mouse/keyboard to rule out a PC problem
    • Ensure that both the mouse and keyboard are showing up in Device Manager
    • Try updating or reinstalling the drivers from within there
    • Replace the mouse/keyboard if all else fails
