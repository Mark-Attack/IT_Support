    • How to Configure a DHCP Scope

    (Windows Server 2022)

    Prerequisites

    Windows Server is Installed

    Steps:

    • click the Windows button and click Server Manager


    • on the top right, click Tools and then click DHCP


    • click the dropdown of the Domain and then right click on IPV4
    • click New Scope and the Scope Wizard will appear


    • Give the Scope a name then click next


    • Configure a range of IP addresses then click next


    • Exclude the addresses that you DO NOT want the server to give out
    • These addresses will be reserved for static IP's
    • click next when completed


    • Set the Lease Duration then click next


    • check the Yes box to configure addition settings
    • click next


    • specify the IP address of the router (default gateway) and add it
    • click next


    • specify the Parent Domain for the clients to use for DNS name resolution
    • click next


    • configure WINS Servers setting if needed
    • click next


    • check Yes to activate the Scope
    • click next


    • click Finish to complete the setup


    • right click the Domain Name and click All Tasks
    • click Restart


    • once completed, go back to IPV4 and view the Scope you created
    • it should say **Active**
