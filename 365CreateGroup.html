    • Creating a Group in Microsoft 365

    Prerequisites

    Logged into M365 admin


    • Click the Users dropdown
    • Click Show all


    • Teams and Groups will appear in the menu
    • It can be pinned to the menu to avoid having to continuously click the show all button


    • Click the Teams and Groups dropdown
    • Click Active Teams and Groups


    • Click Add an Group


    • A popup would appear showing the various types of groups that can be created
    • Chose the appropriate group you wish to create
    • Only the Microsoft 365 option allows a group to be recovered within 30 days of being deleted
    • Click next once completed


    • Give the Group and name and description
    • Click next once completed


    • Assign a group owner by clicking on the Assign Owners button


    • A popup would appear with a list of Users
    • A specific User can also be found by using the search bar
    • Check the box next to the User you wish to be the group owner
    • For multiple owners, check multiple names
    • Click Add once completed


    • The User would appear on the list of owners
    • Click Next to continue


    • Assign group members by clicking on the Add Members button


    • A popup would appear with a list of Users
    • A specific User can also be found by using the search bar
    • Check the box next to the Users you wish to be the group members
    • Click Add once completed


    • The Users would appear on the list of members
    • Click Next to continue


    • Give the group an email address name
    • Set the group privacy setting
    • Anyone can see and join Public groups without any approval
    • Only members can see Private Groups and only Owners can add members
    • Check Microsoft teams if needed
    • Click Next once completed


    • Review all group information previously entered


    • Click Create Group once completed


    • The group would then show up in Active Teams and Groups


    • Groups can be edited or deleted from here
