    • Installing Windows on a Client machine

    (For Organizational Use)

    Prerequisites

    You have permission to install/ reinstall windows on the client machine


    Steps:

    • Download Microsoft Media Creation Tool to a USB and plug it into the client machine
    • restart windows on the client machine and go into BIOS to change the boot order
    • boot into the USB and begin the windows install process
    • choose the correct Language/ Time/ Keyboard then click next


    • click Install now


    • type in a product key if needed then click next


    • chose the appropriate version of windows to install then click next
    • Windows 10/ 11 Pro is usually needed in order to connect to a domain controller


    • accept the license terms then click next


    • chose the Custom install


    • for a clean install of windows, click format
    • select the primary drive then click next


    • wait for Windows to install


    • select a region then click yes


    • confirm keyboard layout then click yes


    • click setup for an organization then click next


    • click Domain join instead


    • type the name of the User then click next


    • create and confirm a password for the User then click next


    • create security questions for the User then click next
    • three questions would be mandatory


    • skip any unnecessary sections
    • wait for windows to finish configuring


    • once windows is completed, right click the start button and click system
    • click Rename this PC
    • rename to the name of the User that will be using this PC
    • click next


    • restart the machine for the changes to take effect
